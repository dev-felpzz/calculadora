🧮 Calculadora Web

Este é o meu primeiro projeto desenvolvido utilizando HTML, CSS e Java.

O objetivo foi praticar lógica de programação, estruturação de layout e organização de código.

🚀 Funcionalidades

Operações básicas (adição, subtração, multiplicação e divisão)

Interface simples e fácil de usar

Estrutura organizada

📚 Sobre o projeto

Este projeto representa o início da minha jornada na programação.
Foi desenvolvido para colocar em prática os conhecimentos que venho estudando e evoluir minhas habilidades como desenvolvedor.
